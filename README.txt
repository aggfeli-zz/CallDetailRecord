Το Project 1 αποτελείται από τα εξής:main.cpp LinkedList.cpp Hashtable.cpp BinaryMaxHeap.cpp
και τα αντιστοιχα headers:LinkedList.h Hashtable.h BinaryMaxHeap.h

Στο αρχείο Hashtable.cpp υπάρχει η υλοποίση του πίνακα κατακερματισμού που αποτελείται από έναν πίνακα δεικτών σε συνδεδεμένη λίστα την LinkedList.
Η κλάση Hashtable περιέχει κατάλληλες συναρτήσεις για την υλοποίηση των εργασιών που απαιτούνται.

Στο αρχείο LinkedList.cpp υπάρχουν οι υλοποιήσεις δύο λιστών!
1) Η LinkedList που είναι η κύρια λίστα που έχει για κόμβους τα buckets hashtable και κάθε κόμβος αποτελείται από το struct Bucket που με την σειρα του περιέχει δείκτη σε επόμενο κόμβο, τον αριθμό του caller/callee και έναν δείκτη στην δευτερεύουσα λίστα
2) Η SimpleList είναι η δευτερεύουσα λίστα που οι κόμβοι της CDR περιέχουν σε έναν πίνακα απο string τις πληροφορίες cdr.

Στο αρχείο BinaryMaxHeap.cpp υπάρχει η υλοποίση του σωρού που ζητήθηκε υλοποιημένο με δείκτες.
Πιο συγκεκριμένα: η κλάση BinaryMaxHeap αποτελείται από δείκτες σε struct Node ρίζας και τελευταίο κόμβο, όπου το struct Node περιέχει δείκτή σε κόμβο γονιό, δείκτες σε κόμβους παιδιών(αριστερό-δεξί) και τον αριθμό του caller αλλα και το ποσό που έχει δαπανήσει.

Στην main με την βοήθεια μιας while διαβάζονται οι διάφορες εντολές από το prompt ή πρώτα από αρχείο αν αυτο έχει δοθεί και με τις κατάλληλες εντολές και μιας switch-case καλούνται οι απαραίτητες συναρτήσεις για την υλοποίηση της κάθε εντολης.

Κατά την εκτέλεση του προγράμματος εμφανίζονται κατάλληλα μηνύματαγια την υλοποίηση ή μη των εντολών αλλα και μηνύματα για εισαγωγή αρχείων όταν αυτό είναι απαραίτητο(tarrif.txt).
